# Contributing to expo-push-notifications-dotnet

Thank you for your interest in contributing to expo-push-notifications-dotnet!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/expo-push-notifications-dotnet.git`
3. Create a branch: `git checkout -b feature/your-feature`

## Development Setup

### Prerequisites

- .NET 8.0, 9.0, or 10.0 SDK
- Git

### Building

```bash
dotnet build
```

### Running Tests

```bash
dotnet test
```

### Code Style

This project uses `.editorconfig` for code style consistency. Please ensure your IDE respects these settings.

Run format check:
```bash
dotnet format --verify-no-changes
```

Fix formatting:
```bash
dotnet format
```

## Pull Request Process

1. Ensure all tests pass
2. Update documentation if needed
3. Add tests for new functionality
4. Follow the existing code style
5. Write clear commit messages

## Reporting Issues

When reporting issues, please include:

- .NET version
- Operating system
- Steps to reproduce
- Expected vs actual behavior
- Error messages (if any)
